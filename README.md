# ms-vendor-lockin-problems

コンピュータはこれほど普及したのに、  
「分からない」「触るのが怖い」「変えられない」  
という感覚は、むしろ強くなっているように見えます。

ExcelやWordは毎日使っているのに、  
中身を説明できる人はいない。  
引き継ぎはうまくいかず、  
詳しい人がいなくなると、業務が止まる。

本リポジトリは、こうした状況が  
**誰かの能力不足ではなく、  
ITの使われ方や教えられ方の“構造”から生まれている**  
という問題意識から作られました。

特定の企業や製品を批判することが目的ではありません。  
操作方法の習得に偏ったIT運用や教育が、  
本来「道具」であるはずのコンピュータを、  
いつの間にか **人を縛る存在**にしてしまっていないか。  

その点を、専門知識のない人にも分かる言葉で整理します。

---

## この資料で扱うこと

- なぜ「分からないまま使われるIT」が増えたのか  
- なぜ文書や仕組みが引き継げなくなるのか  
- なぜ教育や職場で同じ問題が繰り返されるのか  
- そして、どこに立ち戻ればよいのか
