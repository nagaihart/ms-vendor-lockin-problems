# ms-vendor-lockin-problems  
── 考える人間が、再び道具を取り戻すための記録

多くの人が感じている  
「うまくいかない」「理由が分からない」「触れているのに制御できない」

それは個人の能力不足ではない。  
**設計思想の問題**である。

本リポジトリは、  
Microsoft を中心としたベンダーロックイン構造が生み出した  
「考えた過程が残らない世界」から離れ、  

**本来あるべきコンピュータとの関係**を取り戻すための記録である。

---

## このリポジトリの立ち位置

- 操作マニュアルではない  
- 特定製品の比較資料でもない  
- 技術者だけの閉じた議論でもない  

これは、  
**考える人間が、再び道具を主語にするための道筋**を言葉にしたものだ。
